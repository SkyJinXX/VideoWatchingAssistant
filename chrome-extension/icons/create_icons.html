🎤 YouTube语音助手图标生成器

由于Chrome扩展需要图标文件，我们需要创建以下尺寸的图标：

  • icon16.png - 16x16像素（扩展工具栏）
  • icon48.png - 48x48像素（扩展管理页面）
  • icon128.png - 128x128像素（Chrome Web Store）
✅ 图标生成成功！请分别点击下载按钮保存到 icons/ 目录

使用说明：

  1. 点击"生成图标"按钮
  2. 点击"下载所有图标"
  3. 将下载的文件重命名并保存到 icons/ 目录：
    • icon_16x16.png → icon16.png
    • icon_48x48.png → icon48.png
    • icon_128x128.png → icon128.png
  4. 重新加载Chrome扩展
